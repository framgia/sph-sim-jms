# sph-sim-jms

## Setup

1. Clone repository

```
git@github.com:framgia/sph-sim-jms.git
```

2. Install dependencies

- in `web` directory

```
yarn install
```

- in `server` directory

```
yarn install
```

## Local Development

NextJS: http://localhost:3000

```
cd web
yarn dev
```

NestJS: http://localhost:4000

```
cd server
yarn start:dev
```
